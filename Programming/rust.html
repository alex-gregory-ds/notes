14. Rust#

This is designed to be a quick reference for rust.

14.1. Installing Rust and Creating a New Project#

The instructions for install rust can be found here. Once installed, use the cargo new command to create a new project:

cargo new myproject

This creates a directory called myproject which contains a Cargo.toml a src directory which contains a main.rs file. The main.rs files contains a simple hello world function:

fn main() {
    println!("Hello, world!");
}

Use cargo run to run the program:

$ cargo run
   Compiling myproject v0.1.0 (/home/alex/documents/rust/myproject)
    Finished `dev` profile [unoptimized + debuginfo] target(s) in 0.45s
     Running `target/debug/myproject`
Hello, world!

14.2. Mutable and Immutable Variables#

Variables are immutable by default unless the mut keyword is used. Without the mut keyword, in place addition is invalid:

let x = 1;  // Immutable
let mut y = 2;  // Mutable

 x += 1;  // Invalid
 y += 1;  // Valid

14.3. Data Types#

The rust compiler will attempt to infer the type of your variables if a type is not provided. Rust has rules for which data type it defaults to.

let a1 = 10; // 32-bit integer
let a2: i64 = 15; // 64-bit integer

let b1 = 1.0; // 64-bit float
let b2: f32 = 2.0; // 32-bit float

let c1 = true; // bool
let c2 = false; // bool
