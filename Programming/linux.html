9. Linux#

This article contains some random useful information on using Linux.

9.1. The APT Package Manager#

The Advanced Package Tool (APT) is a command-line package manager used in Debian-based Linux distributions like Ubuntu.

9.1.1. Keeping Packages Up to Date#

To update APT’s package index, run the following command:

sudo apt update

This will not update any installed packages, it will only update the package index. To update your installed packages with the new versions in the package index, use the following command:

sudo apt upgrade

9.1.2. Installing New Packages#

New packages can be installed with apt install. For example:

sudo apt install python3.10

You may find that your package is not available in APT’s package indexes in which case you may need to find another package index.

For example, as of writing (2025-05-27), python 3.12 is not available on the APT package index as shown below:

$ sudo apt install python3.12
Reading package lists... Done
Building dependency tree... Done
Reading state information... Done
E: Unable to locate package python3.12
E: Couldn't find any package by glob 'python3.12'

Personal package archives (PPAs) are repositories that allow individuals or teams to share software with others. The Deadsnakes PPA provides recent versions of python including 3.12. You can add the deadsnakes PPA with the add-apt-repository command as follows:

sudo add-apt-repository ppa:deadsnakes/ppa
sudo apt update

Now, sudo apt install python3.12 should work as expected.

This installed htop and two other dependencies libnl-3-200 libnl-genl-3-200. The apt remove htop command will remove all packaged data but leave behind configuration files. The apt purge htop command will remove the configuration files as well.

$ apt list | grep -E "(libnl-3-200|libnl-genl-3-200)"

$ sudo apt remove htop && sudo apt autoremove
